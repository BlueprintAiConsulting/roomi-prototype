WHY MEMORY IS CHEAP

Storage is free. The real memory cost is token injection: each turn sends memory context to Gemini as input tokens. Our compression strategy (7-day full + weekly + monthly + facts) keeps this to ~850 extra tokens/turn, adding ~$9/month at 100 users — instead of ~$45/month for naive full-year injection.
